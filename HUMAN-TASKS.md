# PricePulse — Human Tasks Register

Everything the project needs that cannot be done by code in this repository: client-provided
credentials and infrastructure, third-party account actions, and judgment calls that belong to a
person. Each task lists **who**, **when it blocks what**, and **how to hand the result to the
project**. Status column is meant to be kept up to date in this file.

_Last updated: 10 July 2026 (Phase 0 complete, Milestone 1 in progress)._
